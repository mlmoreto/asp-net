ASP.NET

O ASP.NET é uma plataforma Web que fornece todos os serviços necessários à criação de aplicativos Web baseados em servidor para ambientes corporativos. Ele foi criado em .NET Framework, de modo que todos os recursos do .NET Framework estão disponíveis em seus aplicativos.
